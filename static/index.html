What is this API?

This API provides a RESTful interface for the zambia_geo Python package. It allows developers to query all 10 provinces, list cities/towns by province, and perform global city searches.

BaseUrl

https://zambiageo.pythonanywhere.com

Endpoints

GET /api/v1/provinces

Returns a list of all provinces and their capitals.

[{"name": "Lusaka", "capital": "Lusaka"}, ...]

GET /api/v1/cities/search?q=Kitwe

Search for cities across all provinces.

[{"name": "Kitwe", "population": 522000}]
